# 🔍 Google Search Clone (Vanilla HTML & CSS)

A clean, lightweight recreation of the Google homepage. This project was built to master the fundamentals of layout positioning and to practice replicating a globally recognized UI with high precision.

## 🚀 Key Features
* **Pixel-Perfect UI:** Focused on exact replication of Google's iconic minimalist design, including typography and button styles.
* **Flexbox Centering:** Advanced use of CSS Flexbox to perfectly center the search container both vertically and horizontally.
* **Interactive Elements:** Realistic hover effects on buttons and shadows on the search bar to match the authentic Google user experience.
* **Responsive Design:** Ensures the search bar and navigation links adjust gracefully for mobile and tablet views.

## 🛠️ Tech Stack
* **HTML5:** For semantic and structured markup.
* **CSS3:** For custom styling, including transitions and Flexbox layouts.

## 🧠 What I Practiced
* **Input Styling:** Customizing the default browser search bar to look identical to Google's.
* **Asset Management:** Implementing SVG icons and proper font-family matching.
* **Navigation Layouts:** Building the "header" and "footer" links with proper spacing and alignment.
